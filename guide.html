1

Add the CSS to WordPress

✓ No code

Go to: WordPress Admin → Appearance → Customize → Additional CSS

  1. Open wbl-custom.css in any text editor (Notepad, TextEdit, VS Code)
  2. Select all Ctrl+A → Copy Ctrl+C
  3. Paste into the Additional CSS box in WordPress Customizer
  4. Click Publish

Also add the Google Font (in functions.php or a plugin like WPCode):

// Add to functions.php of your child theme:
function wbl_enqueue_fonts() {
  wp_enqueue_style('wbl-fonts', 'https://fonts.googleapis.com/css2?family=Noto+Sans+Lao:wght@300;400;500;600;700;800;900&family=Space+Grotesk:wght@400;500;600;700&display=swap');
}
add_action('wp_enqueue_scripts', 'wbl_enqueue_fonts');
🔌If you can't edit functions.php, use the free WPCode plugin → add a PHP snippet with the function above.
2

Create the WBL Page in WPBakery

✓ No code
  1. Go to Pages → Add New
  2. Set the title: WBL – ການຮຽນວິຊາຊີບໃນສະຖານປະກອບການ
  3. Set Page Template to Full Width (hides WordPress header/sidebar)
  4. Click "Backend Editor" button at the top
  5. Click "Add Element" → "Raw HTML" then switch to the Text / Code view
  6. Open shortcodes.txt, copy everything between the ==== lines
  7. Paste into the page's code view → Save Draft
  8. Preview to check it looks right → Publish
⚠️Make sure you are in Backend Editor → Text/Code view, NOT the visual editor, when pasting. Pasting into the visual editor will break the shortcodes.

4

Connect the Application Forms (Required)

Developer: 15 min

The forms need a real backend to send emails to phoukhong@gmail.com and save to Google Sheets. Easiest option: Formspree (free).

Option A — Formspree (Free, 5 min setup)

  1. Go to formspree.io → Create 2 free forms
  2. Form 1: "WBL Youth Application" → copy the endpoint URL
  3. Form 2: "WBL Enterprise EOI" → copy the endpoint URL
  4. In the shortcodes.txt (or WPBakery Raw HTML block for the forms), replace YOUR_FORMSPREE_URL with each form's endpoint
  5. In Formspree settings: connect Google Sheets integration → data auto-saves to spreadsheet
<!-- Replace this -->
action="YOUR_FORMSPREE_URL"

<!-- With this (example) -->
action="https://formspree.io/f/xyzabc12"

Option B — WPForms (WordPress plugin, more powerful)

  • Install WPForms → create both forms visually
  • Replace Raw HTML form blocks with WPForms shortcode: [wpforms id="123"]
  • Built-in email notifications + Google Sheets via Zapier

Add to LNCCI Navigation Menu

✓ No code

Link the WBL page from the main LNCCI website navigation (as specified in the TOR).

  1. Go to Appearance → Menus
  2. Find the main navigation menu
  3. Click Pages → select WBL Page → Add to Menu
  4. Label it: WBL ຮຽນວິຊາຊີບ
  5. Save Menu
🔗The TOR requires integration into the existing LNCCI website without affecting the old system. Using a standard WordPress page means zero impact on existing content.
